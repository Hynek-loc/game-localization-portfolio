# game-localization-portfolio
 BG3 Nautiloid – EN → CZ Localization Sample
This repository contains an unofficial localization sample created for
portfolio and professional demonstration purposes only.

- Game: Baldur’s Gate 3
- Developer: Larian Studios
- Language pair: English → Czech
- Tool: Poedit (.po format)

The content is manually selected and partially transcribed to demonstrate:
- narrative localization
- dialogue handling
- UI/system text translation
- tone and register consistency

This is not a mod, not a full localization, and not intended for distribution.
